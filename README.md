# IoT-Weather-Monitoring-System
IoT Weather Monitoring System Using ESP8266, Blynk and ThingSpeak
## PROJECT OVERVIEW:
This project is an IoT-based Weather Monitoring System developed using the ESP8266 NodeMCU and DHT11 temperature and humidity sensor. The ESP8266 collects environmental data and uploads it to both the Blynk IoT platform for real-time mobile monitoring and ThingSpeak for cloud-based data logging and graphical visualization.

The project demonstrates wireless sensor monitoring, cloud connectivity, and real-time visualization using two popular IoT platforms.
## OBJECTIVE:
- Monitor temperature and humidity in real time.
- Display live sensor readings on the Blynk mobile application.
- Store sensor data on the ThingSpeak cloud.
- Visualize environmental data using ThingSpeak graphs.
- Learn IoT concepts using ESP8266 and cloud platforms.

## COMPONENTS REQUIRED:
1. ESP8266 NodeMCU
2. DHT11 Temperature and Humidity Sensor
3. Breadboard
4. Jumper Wires
5. USB Cable
6. Wi-Fi Connection
7. Smartphone with Blynk IoT App
## SOFTWARE REQUIRED:
- Arduino IDE
- ESP8266 Board Package
- Blynk IoT

- Blynk Library

## WORKING PRINCIPLE:
- ESP8266 connects to the Wi-Fi network.
- DHT11 measures temperature and humidity.
- ESP8266 reads the sensor values.
- The data is sent to the Blynk cloud for real-time monitoring.
- The same data is uploaded to ThingSpeak for cloud storage and graphical analysis.
- Users can monitor the readings through the Blynk mobile application and ThingSpeak dashboard.
